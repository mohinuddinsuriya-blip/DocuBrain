# DocuBrain

DocuBrain is an AI-powered document intelligence platform that enables semantic search, document comparison, and automated insight extraction from documents.

## Features
- Semantic document search
- Document comparison
- AI-powered insight extraction
- FastAPI backend with NLP processing
- Interactive frontend interface

## Tech Stack
- Python
- FastAPI
- NLP
- Vector Embeddings
- Streamlit / Web UI

## Installation

1. Clone the repository
git clone https://github.com/mohinuddinsuriya-blip/DocuBrain.git

2. Install dependencies
pip install -r requirements.txt

3. Run backend
uvicorn backend.main:app --reload

4. Run frontend
python frontend/dashboard.py
